Form display with Power animal

For registered users, display welcome message on login. 
If the user has not signed in yet, check if the username, full name and email are provided (all three fields are mandatory).
If they are provided, then user is a valid user and a random power animal is displayed for the user.
If the user has already signed in, shows number of page visits for same session and during 24 hours. 
The page remembers what animal was chosen for the user and shows it again each time they visit the page.
